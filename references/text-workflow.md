# 文本工作流

## 目标

把用户直接输入的一段文本或一段聊天记录总结成一段简短文字。

## 适用场景

- 用户粘贴一段说明、笔记、会议纪要，希望压缩成一段话
- 用户粘贴群聊、IM 对话、沟通记录，希望提炼主要内容

## 推荐步骤

1. 先判断输入更像普通文本还是聊天记录。
2. 普通文本：提取高权重句子，压缩成一段摘要。
3. 聊天记录：识别主要参与者和高权重发言，合成一段总结。
4. 默认直接输出摘要，不走报告模板。
5. 如需入库，把摘要包装成 Markdown 文档，但不落盘到本地。

## 输出要求

- 输出一段完整的话
- 优先保留核心结论、讨论重点、下一步动作
- 不要机械复述所有细节
- 如生成文档，直接上传到 `cms-docdb`
